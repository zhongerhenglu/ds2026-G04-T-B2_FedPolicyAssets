## 摘要

本研究基于 1993—2025 年月度宏观数据与全球资产价格数据，系统识别美联储加息、降息、平稳三大货币政策周期，量化标普 500、纳斯达克 100、黄金、比特币等资产在不同政策环境下的收益、波动与风险特征。研究结果表明：降息周期显著利好权益资产，加息周期对高久期、高成长资产冲击最大；2022 年为 40 年来最激进加息周期，资产波动与回撤均创历史高位；黄金表现由实际利率主导，高通胀加息周期具有抗跌性；比特币样本期较短，结论不具备稳健外推性。研究揭示了美联储政策通过利率→贴现率→全球流动性影响资产定价的传导机制，为跨周期资产配置提供实证参考。

## 一、研究背景

美联储货币政策是全球金融市场最重要的宏观驱动变量。2022—2023 年，美联储为抑制 40 年新高通胀，启动史上最激进加息：联邦基金利率由 0\.25% 快速升至 5\.5%，引发全球股票、债券、大宗商品、加密资产同步剧烈调整。货币政策通过无风险利率、折现因子、汇率与资本流动影响全球资产定价，研究其传导规律具有重要理论与实践价值。

本研究基于权威公开数据，识别 1993—2025 年历次货币政策周期，量化资产收益差异，重点解析 2022 年冲击特征，完成全流程数据科学实证研究。

## 三、数据处理

1. **统一频率**：全部转为月度数据。
2. **收益率**：月度对数收益率。
3. **周期划分**：

   - 平稳（neutral）：其余月份

## 五、核心问题讨论

### 5\.1 加息周期受冲击最大的资产

**结论：长期债券、成长股、比特币最敏感。**
经济学逻辑：加息抬升无风险利率、提高折现率、压低远期资产现值；成长股与比特币久期极高、无稳定现金流，受冲击最强。
**真实数据验证**：2022 年加息周期：比特币最大回撤 \- 77%、纳斯达克 \- 36%、20 年期美债 \- 34%，与理论完全一致。

### 5\.2 黄金加息周期表现是否一致

**结论：不一致，常规偏弱、2022 年例外。**
常规加息：利率上升→持有黄金机会成本上升→金价偏弱。
2022 年：高通胀（PCE 最高 6\.8%）\+ 地缘风险，避险需求对冲利率压力，黄金抗跌。
黄金由**实际利率 = 名义利率−通胀**主导。

### 5\.3 比特币结论是否可靠

**结论：不可靠。**
样本仅 2014 年后、约 10 年，覆盖周期有限；受监管、机构入场、情绪主导，非纯宏观驱动，结论不可外推。

### 5\.4 2022 加息特殊性

**结论：史上最快、最强紧缩。**
12 个月加息 525bp，利率 0%→5\.5%，叠加 40 年高通胀，全球流动性骤紧，资产同步暴跌、波动率飙升，冲击远超历史周期。

## 六、拓展方向

### 6\.1 实际利率测算

`real\_rate = fedfunds − pce\_yoy`
结果输出：`data\_clean/extended\_data\.csv`

### 6\.2 利率敏感性 β

```python
def beta(x, y):
    import numpy as np
    x = x.dropna()
    y = y.dropna()
    return np.cov(x, y)[0, 1] / np.var(x)
```

输出：`data\_clean/beta\_result\.txt`

### 6\.3 VIX 周期情绪分析

输出：`data\_clean/vix\_by\_cycle\.csv`

### 6\.4 拓展代码（main\\\[\_analysis\.py\]\(\_analysis\.py\) 末尾）

```python
def export_extra_results(df):
    df['real_rate'] = df['fedfunds'] - df['pce_yoy']
    import numpy as np
    def beta(x, y):
        x = x.dropna()
        y = y.dropna()
        return np.cov(x, y)[0,1]/np.var(x)
    res = {}
    for a in ['SP500','Nasdaq100','Gold_ETF','Bitcoin']:
        res[a] = round(beta(df['fedfunds'], df[f'{a}_ret']),4)
    with open('data_clean/beta_result.txt','w') as f:
        f.write(str(res))
    df.groupby('cycle')['VIX'].mean().round(2).to_csv('data_clean/vix_by_cycle.csv')
    df.to_csv('data_clean/extended_data.csv')
```

## 七、结论

1. **降息利好权益、加息压制久期资产**，传导效应显著稳定；
2. **2022 年为 40 年最强紧缩冲击**，资产波动创历史高位；
3. **黄金由实际利率主导**，高通胀加息周期具有抗跌性；
4. **比特币样本不足、非宏观主导**，结论不可靠；
5. **美联储政策为全球定价锚**，通过利率→贴现率→流动性传导。
